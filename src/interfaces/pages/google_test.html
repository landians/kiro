Google Login Sandbox

This project page lets you manually walk the Google authorization code flow without creating a separate frontend app.

Start Google Login

Test flow:

  1. Click the button and complete Google consent.
  2. Google redirects back to this project at /auth/google/callback.
  3. The callback page automatically posts the received code to /auth/google/login.
  4. The final JSON response is printed on screen for inspection.
